# E-commerce Sales Analysis (SQL Project)

Author: Ermiyas Hailu  
Tools: PostgreSQL, SQL, Tableau, Excel  
Year: 2026

## Project Description

This project analyzes an e-commerce dataset using SQL in PostgreSQL.

The goal of this project is to perform data validation, KPI analysis,
regional performance analysis, category analysis, and time trend analysis
to answer business questions and prepare the data for dashboard visualization.

## Skills Used

- SQL
- Data Cleaning / Validation
- Aggregation (SUM, COUNT, GROUP BY)
- Date functions (DATE_TRUNC)
- KPI calculations
- Business analysis queries
- Tableau dashboard preparation

## Dataset

E-commerce sales dataset with:
- order_date
- product_name
- category
- region
- quantity
- sales
- profit

## Output

The results of this analysis are used to build a Tableau dashboard.
