# Ecommerce Sales Dashboard (Tableau)

This project analyzes ecommerce sales data using Tableau.

Dashboard features:
- KPI Summary (Sales, Profit, Quantity, Profit Margin)
- Sales Trend Over Time
- Sales by Category
- Profit by Region & Category
- Regional Profit Heatmap

Tableau Public Dashboard:
https://public.tableau.com/app/profile/ermiyas.hailu/viz/Ecommerce_dashboard_17723958111120/Dashboard1

Tools used:
- Tableau Public
- SQL (data preparation)
- CSV dataset
