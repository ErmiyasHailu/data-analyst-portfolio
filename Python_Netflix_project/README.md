# Netflix Data Analysis (Python)

## 📌 Project Overview
This project analyzes the Netflix Titles dataset using Python, pandas, and matplotlib.  
The goal was to clean the data, explore patterns, and generate meaningful insights through visualization.

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Git & GitHub

---

## 📊 Dataset
Netflix Titles Dataset  
(Source: Kaggle – Netflix Movies and TV Shows dataset)

---
## 🔍 Analysis Performed

- Data cleaning (handling missing values)
- Count of Movies vs TV Shows
- Distribution of content ratings
- Titles released per year
- Top countries by number of titles
- Most common genres
- Average duration overall and by type

---

## 📈 Key Insights

- Movies appear more frequently than TV Shows in the dataset.
- Content production increased significantly after 2015.
- The United States has the highest number of titles.
- Drama is among the most common genres.
- Average duration differs between Movies and TV Shows.
